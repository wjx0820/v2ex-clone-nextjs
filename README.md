# v2ex-clone-nextjs

## Stack

- Next.js
- style-components

## Features

- getStaticProps,getStaticPath 每次 build 时预渲染生成静态页面
- revalidate 设置重新抓取新数据的最小时间
- Themes，用 style-components 实现暗黑主题
